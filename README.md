## 🚀 Getting Started

### Prerequisites

* Flutter SDK (>= 3.0.0)
* Android/iOS setup

### Installation

1. Clone the repo:

```bash
git clone https://github.com/Touti-Sudo/QRme.git
cd QRme
```

2. Install dependencies:

```bash
flutter pub get
```


3. Run the app:

```bash
flutter run
```
